# Checklist

## 基础设施
- [x] docker-compose.yml 可一键启动 PostgreSQL、Redis、后端、前端、调度器
- [x] 后端 FastAPI 项目结构清晰，配置通过环境变量加载
- [x] 数据库迁移可正常执行（alembic upgrade head）

## 用户认证与角色
- [x] 注册接口对邮箱格式校验、密码使用 bcrypt 哈希存储
- [x] 登录接口签发 JWT，后续请求可通过 JWT 鉴权
- [x] 角色守卫生效：学员访问 admin 接口返回 403
- [x] 被禁用用户无法登录

## 课程内容系统
- [x] 管理员可对课程/章节进行增删改查
- [x] 学员可浏览课程列表与章节详情
- [x] 章节正文以 Markdown 存储，前端可正常渲染

## 靶机模板库
- [x] 管理员可新增/编辑/下架靶机模板
- [x] 学员查询靶机列表/详情时不返回 Flag 字段
- [x] 模板包含名称、描述、难度、镜像、暴露端口、Flag

## 靶机实例调度
- [x] 学员启动靶机后获得隔离 Docker 容器与可访问的端口
- [x] 同一学员对同一靶机并发实例数受限于配置（默认 1）
- [x] 学员可主动销毁实例
- [x] 超过存活上限（默认 60 分钟）的实例被自动回收
- [x] 容器销毁后端口映射被释放

## Flag 提交与积分
- [x] 提交 Flag 与预期值比对，返回是否正确
- [x] 首次正确提交记录通关并发放积分，重复提交不重复发分
- [x] 通关记录可查询

## 个人中心与统计
- [x] 学员个人中心展示通关数、积分、最近活动
- [x] 管理员统计接口返回用户数、靶机使用数、通关排行

## 后台管理
- [x] 管理员可查看/禁用用户
- [x] 管理员可管理课程与靶机模板
- [x] 管理后台接口统一在 admin 路由前缀并受角色守卫保护

## 前端界面
- [x] 登录/注册页可用
- [x] 课程列表与章节阅读页可用
- [x] 靶机列表、启动/销毁、Flag 提交页可用
- [x] 个人中心页可用
- [x] 管理后台页（用户/课程/靶机/统计）可用
- [x] 页面响应式，适配桌面与移动端

## 端到端验收
- [x] 提供至少一个示例靶机镜像用于联调
- [x] 完整流程跑通：注册→登录→学习→启动靶机→提交 Flag→查看积分
